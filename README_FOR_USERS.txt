===============================================
  AutoResponder1000 - Quick Start Guide
===============================================

FIRST TIME SETUP:
-----------------
1. Copy "responses.template.json" and rename it to "responses.json"
   (Keep them in the same folder as AutoResponder1000.exe)

2. Run AutoResponder1000.exe

3. Click "Settings" button

4. Change "[Your Name]" to your actual name in the Default Signature

5. Update email addresses in WB Report section if needed

6. Click "Save"


HOW TO USE:
-----------
STRICKLAND CLOCK INS:
  - Paste email subject line
  - Click "Copy Clock In Response"
  - Paste into your email

SERVICE CHANNEL:
  - Enter rate and NTE
  - Click "Add Shift(s) and Generate Response"
  - Enter shift details
  - Click "Copy Response"

SEND WORK ORDER (PIN):
  - Enter PIN number
  - Click copy button

RATE APPROVAL:
  - Enter rate
  - Click copy button

WB REPORT:
  - Select date
  - Click "Create WB Report Email"


TROUBLESHOOTING:
----------------
If you get errors:
- Make sure "responses.json" exists in the same folder as the .exe
- Check that the folder is not read-only
- Contact your IT support if issues persist


YOUR SETTINGS:
--------------
All your settings are saved in "responses.json"
Keep this file safe - it contains your personalized templates!
